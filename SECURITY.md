# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project (e.g., an unintended way the platform could be exploited outside a lab), please **do not open a public GitHub issue**.

Instead, email the maintainer directly (or open a private GitHub security advisory).

---

## Responsible Use

This project is a **Breach and Attack Simulation (BAS)** research tool. It contains:
- Real attack technique implementations mapped to MITRE ATT&CK
- PowerShell remoting scripts
- C2 framework integration (Sliver)

### Acceptable Use
- Isolated lab environments (VirtualBox Host-Only networks)
- Authorized red team assessments
- Academic and cybersecurity research

### Prohibited Use
- Any system you do not own or have **explicit written authorization** to test
- Production, corporate, or cloud infrastructure
- Any activity that violates local, national, or international law

---

## Supported Versions

This project is research software. Only the latest commit on `main` is maintained.

| Version | Supported |
|---------|-----------|
| Latest (`main`) | ✅ |
